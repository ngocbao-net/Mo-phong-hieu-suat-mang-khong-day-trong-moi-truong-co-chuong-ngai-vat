Important interpretation note
=============================
- Truth-based LOS/NLOS speed/path-loss figures describe the physical separation between LOS and NLOS.
- Downstream MAE figures quantify how much error kNN / kNMAP introduce when their predicted labels are used to derive path loss / speed.
- Therefore, claims about "kNMAP gives better downstream maps" should be supported by the MAE plots, not only by truth-based LOS/NLOS averages.
